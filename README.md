Распознавание геометрических фигур с помощью CNN
Проект реализует модель глубокого обучения для классификации простых геометрических фигур (круг, квадрат, треугольник) на изображениях.

Описание
Модель представляет собой свёрточную нейронную сеть (CNN), обученную на синтетических данных. Подходит для:

образовательных целей;

прототипирования систем компьютерного зрения;

задач базовой классификации форм.

Функциональность
Генерация синтетических обучающих данных (круги, квадраты, треугольники).

Обучение CNN-модели с визуализацией процесса.

Предсказание класса фигуры на новых изображениях.

Оценка качества модели (матрица ошибок, отчёт по классификации).

Архитектура модели
Свёрточные блоки:

3 слоя Conv2D (32→64→64 фильтров, ядро 3×3, ReLU).

2 слоя MaxPooling2D (2×2) для понижения размерности.

Классификатор:

Flatten → Dense(64, ReLU) → Dropout(0.5) → Dense(3, softmax).

Вход: 64×64×1 (оттенки серого).
Выход: 3 класса (вероятности).

Требования
Установите зависимости:

bash
pip install tensorflow opencv-python scikit-learn matplotlib seaborn numpy
Запуск
Обучение модели:

bash
python train.py
Сгенерирует данные, обучит модель и сохранит её как shape_classifier_model.h5.

Предсказание на своём изображении:

bash
python predict.py --image path/to/your/image.jpg
Замените path/to/your/image.jpg на путь к вашему файлу.

Структура проекта
train.py — обучение модели на синтетических данных.

predict.py — предсказание класса фигуры на новом изображении.

shape_classifier_model.h5 — сохранённая обученная модель (создаётся после запуска train.py).

Результаты
После обучения выводятся:

График точности и потерь.

Матрица ошибок.

Отчёт по метрикам (precision, recall, F1-score).

Визуализация предсказаний на тестовых примерах.
