# Распознавание геометрических фигур
## Веб‑приложение для распознавания геометрических фигур.

**_Описание_**
Проект представляет собой веб‑сервис, который:
- принимает изображение геометрической фигуры от пользователя;
- анализирует его с помощью модели машинного обучения;
- возвращает результат классификации (название фигуры).

**_Функциональные возможности:_**
Загрузка изображения через веб‑интерфейс.
Классификация геометрических фигур по фото.
Отображение результата с указанием вероятности.
