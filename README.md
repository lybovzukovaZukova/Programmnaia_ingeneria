# Распознавание геометрических фигур
## Веб‑приложение для распознавания геометрических фигур.

**_Описание_** <br>
Проект представляет собой веб‑сервис, который:<br>
- принимает изображение геометрической фигуры от пользователя;<br>
- анализирует его с помощью модели машинного обучения;<br>
- возвращает результат классификации (название фигуры).<br>
<br>
**_Функциональные возможности:_** <br>
Загрузка изображения через веб‑интерфейс.<br>
Классификация геометрических фигур по фото.<br>
Отображение результата с указанием вероятности.<br>
